<div align="center">

A comprehensive Tailwind CSS learning playground built with **React**, **Vite**, and **Tailwind CSS**.

Explore, learn, and experiment with Tailwind utility classes through interactive examples and organized topic-based demonstrations.

</div>

---

## About

Tailwind CSS Playground is a frontend learning project designed to help developers understand and practice Tailwind CSS utilities through real-world examples.

The project organizes Tailwind concepts into dedicated sections, making it easier to explore utility classes, layouts, responsive design techniques, and styling patterns commonly used in modern web applications.

## Installation

Clone the repository:

```bash
git clone https://github.com/rahulraji18/tailwind-css-playground.git
```

Navigate to the project directory:

```bash
cd tailwind-css-playground
```

Install dependencies:

```bash
npm install
```

---

## Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
